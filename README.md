# FIFA World Cup Performance Dashboard

An interactive **Power BI Dashboard** that visualizes historical FIFA World Cup data, providing insights into team performances, player statistics, and match outcomes from 1930 to 2014.

## 📊 Project Overview
The goal of this project is to transform raw Excel data into a visual story. This dashboard allows users to explore the evolution of the World Cup, identifying trends in goal scoring, attendance, and national dominance.

### Key Insights:
* **Winner Analysis:** Visualizing which countries have the most titles and podium finishes.
* **Goal Trends:** Tracking the average goals per game across different eras.
* **Match Statistics:** Analyzing home vs. away team performance.
* **Attendance:** Identifying the most popular tournaments and venues.

## 📁 Repository Structure
* `FIFA DASHBOARD.pbix`: The primary Power BI Desktop file containing the data model and visualizations.
* `WorldCups.xlsx`: Summary data for every World Cup tournament.
* `WorldCupMatches.xlsx`: Detailed results for every match played.
* `WorldCupPlayers.xlsx`: Player-level data for lineups and performance.

## 🚀 How to Use
1.  **Clone the repo:**
    ```bash
    git clone https://github.com/moha0808/FIFA-DASHBOARD.git
    ```
2.  **Open the Report:** Ensure you have [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed. Open the `FIFA DASHBOARD.pbix` file.
3.  **Data Refresh:** If the data paths break, point the Data Source Settings to the `.xlsx` files included in this repository.

## 🛠️ Tools Used
* **Power BI:** Data modeling, DAX measures, and visualization.
* **Power Query:** Data cleaning (handling nulls, formatting dates, and merging tables).
* **Excel:** Source data storage.
